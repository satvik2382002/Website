# Personal Website

A simple static website built using HTML and CSS.  
This project demonstrates basic front-end skills such as layout design, navigation, and page structure.

Tech Stack
- HTML
- CSS

 Features
- Multi-page website
- Navigation menu
- Image gallery
- About and Contact pages
- Basic UI layout

 Project Purpose
This project is created to showcase basic front-end development .  
It is suitable for beginners and helps to demonstrate knowledge of deploying this on AWS.
